# PaletteGen

PaletteGen is a modern color palette generator built with HTML, CSS, and JavaScript. It helps designers, developers, and creators generate beautiful color combinations instantly for websites, applications, branding, and UI/UX projects.

##  Features

* Generate random color palettes instantly
* Copy HEX color codes with one click
* Clean and responsive interface
* Modern glassmorphism design
* Fast and lightweight
* Mobile-friendly experience
* Easy to use for designers and developers

##  Technologies Used

* HTML5
* CSS3
* JavaScript
* Glassmorphism UI Design
